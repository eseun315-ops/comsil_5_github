로컬저장소 생성:git init
복제: git colne(이거 하면 자동으로 연결됨)
원격저장소 연결: git remote add origin "주소"
로컬에 파일 추가: git add(생성 아님)
변경한 내용 코멘트: git commit -m "~~"
원격저장소에 반영: git push origin master
반대: git pull (origin master까지 하는게 좋음)
